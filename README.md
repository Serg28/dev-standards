# Репозиторий стандартов разработки

Этот репозиторий содержит набор инструкций, правил и лучших практик для поддержания высокого качества кода и обеспечения согласованности в процессе разработки.

### Для разработчиков

*   **[Лучшие практики](./Developer/best_practices.md)**: Рекомендации по написанию чистого, эффективного и поддерживаемого кода.
*   **[Чистый код](./Developer/clean_code.md)**: Принципы и примеры для улучшения читаемости и структуры кода.
*   **[Соглашения по именованию](./Developer/naming_conventions.md)**: Правила именования переменных, функций, классов и других сущностей.
*   **Оптимизация**
    *   **[Оптимизация Artisan-команд](./Developer/optimization/artisan-commands-optimization.md)**: Инструкция по предотвращению нагрузки на сервер из-за некорректного использования конструкторов команд.

*   **Оптимизация**
